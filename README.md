# My Home Workout

A mobile-friendly workout tracker for a beginner home gym program — months 1–2.

## Features

- **A/B workouts** — Pull+Biceps+Core (A) and Push+Triceps+Legs (B)
- **8-week progression table** — gradual intensity increase with guidance
- **Expandable exercise tips** — tap any exercise for form cues
- Blood sugar monitoring note for diabetic athletes

## Local dev

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
